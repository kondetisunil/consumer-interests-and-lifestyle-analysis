# youtube-engagement-strategy
How many people liked/commented per view

This project applies SQL to social media metadata to identify the "conversion velocity" of digital content.

The Strategy:

Engagement vs. Reach: Instead of just looking at high view counts (Vanity Metrics), I calculated the Engagement Rate. This identifies which content categories create the most "loyal" users.

Scaling Logic: By analyzing the conversion_velocity, I identified the specific video types that drive the highest ROI for subscriber growth.

This framework mirrors the operational logic I used at the Elevance Health account to identify high-performing partners and optimize resource allocation.
